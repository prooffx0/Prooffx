# ProofX · LUDI Verifier  
“We protect truth so humans can survive the age of synthetic lies.”

---

## 1. What is ProofX?

**ProofX is a trust layer for the AI era.**  
우리는 이미지, 문서, 계약서, 메시지, 영상 등 “생성된 모든 것”에 **위조 불가능한 생성 시점 기록 (Immutable Origin Proof)** 을 남기고, 이후에 누구나 그 진위를 검증할 수 있게 합니다.

이 트러스트 레이어의 심장부가 바로 **LUDI Verifier** —  
“이거 진짜야?”라고 물었을 때 답해주는 공개 검증자입니다.

> ProofX = 기록  
> LUDI Verifier = 검증  
> → Together: a defense system for human truth.

## 2. The Problem: “What’s real?”

우리는 지금, 디지털에서 “그게 진짜야?”라는 질문이 너무 쉽게 나오는 시대에 살고 있습니다.

- 보이스피싱, 계정 사칭, 위조된 송금 요청  
- 조작된 대화 캡처와 존재하지도 않은 약속을 실제 합의처럼 보이게 만든 메시지  
- 한 사람의 말투와 목소리까지 복제해서 “그 사람이 그렇게 말했다”처럼 연출하는 합성 음성과 생성형 AI

공격은 점점 더 자동화되고, 대량화되고 있습니다.  
하지만 방어는 아직도 개인이 “그건 조작된 거예요”라고 설명하러 다니는 수준에 멈춰 있습니다.

이건 단순히 “사기꾼이 교묘해졌다”의 문제가 아닙니다.  
진짜 문제는, 우리가 서로를 믿을 수 있는 **공통 기준**이 사라지고 있다는 것입니다.

- 이 스크린샷이 실제로 저 사람이 보낸 메시지가 맞는지  
- 이 광고 사진이 진짜 그 제품에서 찍힌 원본인지  
- 이 계약서가 중간에 고쳐지지 않았는지  
- 이 영상이 실제 상황인지 아니면 편집본인지

우리는 점점 더 자주 의심하게 되고,  
그 의심은 결국 사람들 사이의 신뢰 자체를 갉아먹습니다.

ProofX와 LUDI Verifier는 바로 그 지점에서 시작합니다.

우리가 만들고 싶은 건 ‘불신 사회의 감시 도구’가 아니라  
**깨끗한 연결고리**입니다.

- 어떤 디지털 정보가 “처음 어떻게 만들어졌는지(원본)”  
- 그 이후 “어떤 수정 과정을 거쳤는지(이력)”  
- 그리고 “지금 눈앞의 버전이 그 히스토리와 일치하는지(검증)”

이 세 가지를 투명하게 열어놓으면  
사람은 서로를 의심 없이 다시 이야기할 수 있습니다.

우리는 “넌 나를 믿어야 해”라고 강요하려는 게 아닙니다.  
우리는 “확인해봐, 그대로야”라고 말할 수 있는 환경을 만들려 합니다.

ProofX와 LUDI Verifier의 목표는  
세상을 더 무섭게 만드는 게 아니라,  
사람들 사이에 다시 안심하고 말할 수 있는 공간을 복구하는 것입니다.

## 3. Founder Note

이건 추상적인 문제의식에서 나온 프로젝트가 아닙니다.

나는 실제로 디지털 사기 상황에 노출된 적이 있고, 그 과정에서 한 가지를 굉장히 명확하게 보게 되었습니다.

피해자는 이미 흔들린 상태에서 시작합니다.  
그런데도 ‘사실관계 증명’의 책임은 피해자 쪽에 올라옵니다.

- “정말 그런 연락을 받았다는 걸 증명할 수 있나요?”  
- “그 송금 요구가 조작된 거라는 근거는요?”  
- “당신이 속았다는 걸 우리가 어떻게 확신하죠?”

반면 공격은 개인이 아니라 시스템으로 움직입니다.  
해외 조직은 사람을 유인해서 강제로 사기 시스템에 투입시키고, 합성 음성·가짜 계약·조작된 로그를 공장식으로 만들어냅니다.  
이건 한 명의 사기꾼이 아니라, 산업화된 속임수입니다.

그리고 이런 조작은 더 이상 어둡고 먼 범죄 구역 안에만 있지 않습니다.

- 실제 결제/인증 증빙처럼 보이지만 생성형 이미지로 만들어진 위조 내역

우리는 더 이상 화면을 보고 직관적으로 “저건 증거네”라고 믿을 수 없는 상태까지 와 있습니다.

ProofX는 “불신을 증폭시키는 도구”가 아니라 그 반대입니다.

우리가 하려는 건, 사람과 사람 사이에 다시 쓸 수 있는 **안전한 신뢰의 언어**를 만드는 것입니다.

- 내가 실제로 본 것 / 들은 것 / 동의한 것 / 전달한 것  
- 그 ‘원본’과 ‘변경 이력’을 남기고  
- 제3자가 확인 가능한 형태로 꺼내서  
- “이건 진짜 이렇게 만들어졌고, 여기까지 이렇게 바뀌었다”를 투명하게 보여주는 것

우리는 “사람을 완벽하게 만들겠다”고 약속하지 않습니다.  
우리는 “거짓으로 사람을 부수는 방식”을 훨씬 어렵게 만드는 인프라를 만들고자 합니다.

ProofX의 목표를 한 줄로 요약하면 다음과 같습니다:

**디지털 환경에서 — 그리고 앞으로 사람 대신 의사결정을 수행할 AI 에이전트까지 포함해서 —  
누구든 자기 현실을 투명하게 증명할 수 있게 하는 것.**

## 4. The Solution

### 4.1 ProofX — The Trust Layer  
ProofX는 인터넷 위에 얹히는 **신뢰 인프라 계층(Trust Layer)** 입니다.

- 어떤 컨텐츠(이미지, 문서, 음성, 대화 로그 등)가 “처음으로 생성된 순간”의 원본 상태를 캡처합니다.  
- 그 상태를 **불변 원본 스탬프 (Immutable Origin Stamp)** 로 블록체인에 고정합니다.  
- 그 기록은 삭제, 편집, 레트로액티브 수정이 불가능합니다.

한 문장으로 말하면:

> “원본이 언제 어떤 상태로 만들어졌는지”를  
> 거짓말로 다시 쓸 수 없게 만든다.

이건 앞으로의 인터넷/AI 환경에서 기본 생존장치가 됩니다.

### 4.2 LUDI Verifier — The Public Verifier  
LUDI Verifier는 ProofX 위에서 작동하는 공개 검증자입니다.

누구나 어떤 데이터/파일/발언을 가져와서 물을 수 있습니다:

- “이건 ProofX에 최초로 찍힌 상태와 동일합니까?”  
- “중간에 변조 이력이 있습니까?”  
- “이 말/이미지/기록은 신뢰 가능한 발신자로부터 나왔습니까?”

LUDI는 “신용을 개인이 구걸하지 않도록” 설계되었습니다.  
(= 피해자가 ‘제발 믿어주세요’라고 빌지 않아도 되게)

기술적으로는 “원본 증명 + 무결성 검증 + 출처 신뢰도 평가”를 수행하는 모듈이며,  
비즈니스적으로는 API / SDK / 파트너 연동의 핵심입니다.
